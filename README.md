# ARC Raiders Event Timers

Веб-приложение для отображения таймеров событий игры ARC Raiders, построенное на Next.js.

## Технологии

- **Next.js 14** - React фреймворк
- **React 18** - UI библиотека
- **GitHub Pages** - хостинг

## Локальный запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите dev-сервер:
```bash
npm run dev
```

3. Откройте браузер и перейдите по адресу: http://localhost:3000

4. Для сборки production версии:
```bash
npm run build
```

5. Для запуска production версии локально:
```bash
npm start
```


## Структура проекта

- `app/` - страницы и роутинг Next.js
- `components/` - React компоненты
- `hooks/` - кастомные хуки
- `utils/` - утилиты и вспомогательные функции
- `services/` - сервисы для работы с API
- `public/` - статические файлы

## Описание

Приложение получает данные о событиях ARC Raiders с API и отображает их на русском языке с указанием времени проведения. Активные события подсвечиваются зелёным цветом. 

Доступна фильтрация событий, группировка по событиям и картам, отображение в виде аккордеонов для удобной навигации.
